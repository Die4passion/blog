### 类型转换

```js
"4px"-2 = NaN
```
> 使用+ 将非数字转为数字,等同于Number(...)

```js
 +apples + +oranges  //两个数字字符串相加
 
 alert( 4 ** (1/2) ); // 2 (1 / 2 幂相当于开方，这是数学常识)
 alert( 8 ** (1/3) ); // 2 (1 / 3 幂相当于开三次方)
```
