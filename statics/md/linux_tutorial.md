# Linux Tutorial by [Ryan's](https://ryanstutorials.net/)

### 1. vim 操作

- `h` 左 `j` 下 `k` 上 `l` 右

- `0` 跳到行开头

- `^` 本行开头不为空格的位置

- `$` 本行结尾

- `nG` 第n行

- `gg` 第一行

- `w` 下一个单词开头

- `b` 上一个单词开头

- `{` 向后移动一段

- `}` 向前移动一段

- `ctrl + o` 跳转回之前的位置

- `ctrl + i` 返回跳转之前的位置

- `x` 删除单个字符

- `nx` 删除n个字符

- `dd` 删除当前行

- `dn` d后面跟一个移动命令 删除命令到的地方 比如`d5w` 删除5个单词

- `u` 撤销最后一个动作

- `U` 撤销对当前行的所有更改 `ctrl + r`

- `/some text`  搜索 `n` 下一个 `N` 上一个
